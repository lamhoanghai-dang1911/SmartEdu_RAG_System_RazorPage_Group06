# SmartEdu Razor Web - Setup and Run Guide

## Prerequisites

Before running the project, make sure the following software is installed on your machine:

- **Visual Studio 2022** (recommended)
- **.NET SDK** (version compatible with the project, preferably .NET 8 or later)
- **SQL Server** or **SQL Server Express**
- **SQL Server Management Studio (SSMS)** (optional but recommended)

## Project Structure

The solution consists of the following projects:

- `SmartEdu.RazorWeb` - Main Razor Pages web application.
- `SmartEdu.Business` - Business logic layer.
- `SmartEdu.DataAccess` - Data access layer.
- `SmartEdu.Shared` - Shared models and utilities.
